# Frontend

### Installation Frontend ONLY
1. Enter your base URL in `.env`
   ```
   REACT_APP_BASE_URL = 'ENTER YOUR BACKEND URL';
   ```
2. Install and run
   
   i. Using Docker
      ```
      docker build -t va_finder_frontend .
      docker run -it va_finder_frontend
      ```
   ii. Using npm
      ```
      npm install
      npm start
      ```
